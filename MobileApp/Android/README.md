# Приложение под Android

Приложение по Android позволяет на устройстве под управлением системы Android смотреть общую и ежедневную статистику.

Приложение имеет три раздела: общая статистика, ежедневная статистика и настройки. 
Переход по разделам приложения происходит через боковое левое меню. Меню открывается проведением пальцем от левого края или по нажатию на кнопку (три горизонтальных линии) в верхнем левом углу.

Первым делом, необходимо в настройках указать адрес сервиса.

#### Порядок просмотра статистики:
* Выбирать сайт (для ежедневной так же выбирается личности и период).
* Нажать на круглую кнопку, на которой изображена галочка.
* Чтобы изменить свой выбор, необходимо нажать на круглую кнопку, на которой изображена шестерёнка.
